# 🔐 Auth Microservice (Spring Boot + Keycloak)

This repository contains an **Authentication and Authorization microservice** built with **Spring Boot**, **Keycloak**, and **PostgreSQL**.  
It provides APIs for **user registration** and **login** with JWT-based authentication, and is fully containerized with **Docker Compose**.

---

## ✨ Features

- ✅ User Registration (`/auth/register`)
- ✅ User Login (`/auth/login`)
- ✅ Role assignment on registration
- ✅ JWT-based authentication (via Keycloak)
- ✅ PostgreSQL persistence for Keycloak
- ✅ Dockerized deployment

---

## 🛠 Tech Stack

- [Spring Boot 3.x](https://spring.io/projects/spring-boot)  
- [Spring Security (OAuth2 Resource Server)](https://spring.io/projects/spring-security)  
- [Keycloak](https://www.keycloak.org/) – Identity and Access Management  
- [PostgreSQL](https://www.postgresql.org/)  
- [Docker Compose](https://docs.docker.com/compose/)
